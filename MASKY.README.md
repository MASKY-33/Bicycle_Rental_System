# Bicycle Rental System
This system allows the user to search for a bicycle type and instantly see its rental price.
The program keeps running until the user types "quit", making it suitable for continuous rental desk usage.

## Features
- Stores bicycle types and rental prices in a dictionary
- Lets the user search any bicycle by name
- Checks if the bicycle exists in the system
- Shows the correct rental price when found
- Gives a clear message if the bicycle is not available
- Runs in a continuous loop until the user types "quit"

## How to use
Run the system and type the name of any bicycle.
The program will display the rental price.
Type "quit" to exit the system.
